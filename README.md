# ⚡ DevStash

> A blazing-fast collection of developer utilities — all running entirely in your browser.  
> No signups, no backend.
> https://cozy-paprenjak-d54d4d.netlify.app/
---

## 🛠️ Tools

| Tool | Description |
|------|-------------|
| **JSON Formatter & Validator** | Format, minify, and validate JSON with error highlighting |
| **Base64 Encoder/Decoder** | Encode text to Base64 or decode Base64 back to text |
| **URL Encoder/Decoder** | Percent-encode or decode URL strings instantly |
| **Regex Tester** | Test regular expressions with live match highlighting |
| **Color Converter** | Convert between HEX, RGB, and HSL with a live color swatch |
| **Text Diff Viewer** | Compare two blocks of text and highlight differences line by line |
| **Hash Generator** | Generate SHA-256 hashes using the browser's native Web Crypto API |
| **Word & Character Counter** | Live count of words, characters, sentences, and reading time |
| **Markdown Previewer** | Side-by-side editor with live rendered preview |
| **Unix Timestamp Converter** | Convert Unix timestamps to readable dates and back |
| **Number Base Converter** | Convert between binary, octal, decimal, and hexadecimal |

## ✨ Features

- **100% client-side** — nothing is ever sent to a server
- **Zero dependencies** — no npm, no frameworks, no build step required
- **Zero cost** — free to use, free to host, free to fork
- **Mobile responsive** — works on any screen size
- **Fast** — loads in under 2 seconds

---

## 🚀 Getting Started

No installation needed. Just clone and open:

```bash
git clone https://github.com/MaxelXA/devstash.git
cd devstash
```

Then open `index.html` in any browser. That's it.

---

## 🌐 Deployment

Static site — deploy anywhere for free:

- **Vercel** — connect your GitHub repo, deploys automatically on every push
- **Netlify** — drag and drop the project folder at netlify.com/drop
- **GitHub Pages** — enable it under repo Settings → Pages → Deploy from branch

---

## 🧰 Tech Stack

- HTML5 + CSS3 (custom dark theme with CSS variables)
- Vanilla JavaScript (ES6+)
- [marked.js](https://marked.js.org/) via CDN for Markdown rendering
- Web Crypto API for SHA-256 hashing

No build tools. No package.json. Open the file, it works.
